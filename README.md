## Python Practice Questions

A collection of Python practice questions and solutions covering loops, functions, OOP, and more.
These exercises are designed to improve problem-solving skills and reinforce Python basics and advanced concepts.

## Project Structure

loops/ – While, for, and nested loop exercises

functions/ – Function-related exercises

oop/ – Object-Oriented Programming exercises (Inheritance, Polymorphism)

examples/ – Real-world simulation exercises (ATM, taxi meter, traffic light, dice roll)

## Features

Looping through lists, strings, and numbers

User input validation with loops

Nested loops for patterns and combinations

Functions for calculations, filtering, and string manipulation

OOP exercises: Inheritance, Polymorphism, Method Overriding

Real-world simulation programs: ATM PIN check, Taxi meter, Traffic light, Dice roll

Practical examples: Email filtering, product pricing, hashtags extraction, password masking

## Examples of Practice Questions

Loop through emails and print only @gmail.com

Bus seating chart generation
